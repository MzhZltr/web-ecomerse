## 📋 Deskripsi

TokoEcom adalah website e-commerce sederhana yang dibangun dengan HTML, CSS, dan JavaScript murni. Website ini menampilkan berbagai produk fashion dengan fitur keranjang belanja, filter kategori, dan dark mode.

## ✨ Fitur

- ✅ Navbar responsif dengan hamburger menu
- ✅ Halaman produk dengan grid layout
- ✅ Filter produk berdasarkan kategori
- ✅ Shopping cart dengan localStorage
- ✅ Form kontak dengan validasi
- ✅ Dark mode toggle
- ✅ Smooth scroll
- ✅ Typing effect pada hero section
- ✅ Scroll progress bar
- ✅ Visitor counter
- ✅ Toast notifications
- ✅ Responsive design (Mobile, Tablet, Desktop)

## 🛠️ Teknologi

- HTML5
- CSS3 (Flexbox, Grid, CSS Variables)
- JavaScript (ES6+)
- LocalStorage API
